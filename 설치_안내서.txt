====================================
JS일렉트로닉 ERP 시스템 v1.0.0
설치 및 사용 안내서
====================================

[설치 방법]

1. macOS 사용자:
   - "JS일렉트로닉 ERP-1.0.0-arm64.dmg" 파일을 더블클릭
   - 앱을 Applications 폴더로 드래그
   - 첫 실행 시 보안 경고가 나오면 "열기" 선택

2. Windows 사용자:
   - Windows용 설치 파일은 준비 중입니다
   - 임시로 macOS에서 사용해주세요

[실행 방법]
1. 설치 후 "JS일렉트로닉 ERP" 아이콘 클릭
2. 자동으로 서버가 시작되고 웹 브라우저가 열립니다
3. 포트 3001에서 실행됩니다

[주요 기능]
- 매출 관리: Excel 파일 업로드 및 자동 분석
- 거래처 관리: 고객사별 매출 추적
- 거래명세서: PDF/Excel 양식 자동 생성
- 매출 분석: 차트를 통한 시각화
- 자동 백업: 매일 새벽 2시 자동 백업

[파일 위치]
- 데이터베이스: database/erp.db
- 백업 파일: backups/ 폴더
- Excel 업로드: uploads/ 폴더

[문의사항]
기술 지원이 필요하시면 연락 주세요.
