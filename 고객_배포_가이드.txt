JS일렉트로닉 ERP 시스템 v3.0
설치 가이드

📁 받으신 파일 목록:
1️⃣ JS_ERP_Windows_FIXED_v3_FINAL.zip (1.2MB) - 프로그램 파일

🚨 중요: 설치 순서를 반드시 지켜주세요!

====================================
1단계: 기존 프로그램 완전 제거 (필수)
====================================

만약 이전에 설치한 ERP 프로그램이 있다면:

1. uninstall_complete.bat 파일을 우클릭하세요
2. "관리자 권한으로 실행"을 선택하세요
3. 경고 메시지가 나오면 아무 키나 누르세요
4. 7단계 삭제 과정을 기다리세요
5. 완료 후 컴퓨터를 재부팅하세요

⚠️ 주의: 반드시 관리자 권한으로 실행해야 합니다!

====================================
2단계: 새 프로그램 설치
====================================

1. Node.js 설치 (한 번만 필요):
   - https://nodejs.org 접속
   - LTS 버전 다운로드 및 설치
   - 설치 후 컴퓨터 재부팅

2. ERP 프로그램 설치:
   - JS_ERP_Windows_FIXED_v3_FINAL.zip 압축 해제
   - start_en.bat 파일을 우클릭
   - "관리자 권한으로 실행" 선택
   - 설치 완료까지 3-5분 대기
   - 브라우저에서 자동으로 ERP 실행 확인

====================================
설치 완료 확인
====================================

설치가 정상적으로 완료되면:
✅ 브라우저가 자동으로 열립니다
✅ "JS일렉트로닉 ERP" 제목이 보입니다
✅ 대시보드 화면이 정상 표시됩니다
✅ 매출 데이터가 로드됩니다

====================================
문제 발생 시
====================================

❌ 백지 화면이 나올 경우:
   → troubleshoot.bat 실행

❌ 설치가 중단될 경우:  
   → 1단계를 다시 실행 후 재부팅

❌ 포트 오류가 날 경우:
   → 다른 프로그램 종료 후 재시작

❌ 그래도 안 될 경우:
   → 고객센터 연락 (연락처: 별도 안내)

====================================
사용 방법
====================================

일상 사용 시:
1. 바탕화면 "JS ERP" 바로가기 클릭
2. 브라우저에서 자동으로 ERP 실행
3. 작업 완료 후 브라우저 창만 닫기

⚠️ 주의: 검은색 명령창은 닫지 마세요!
